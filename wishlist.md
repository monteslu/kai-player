# Kai Player Wishlist

## Format Support Expansion
- **CDG Support** - Industry standard CD+G karaoke format
- **MP3+G Support** - Popular compressed karaoke format  
- **KAR Support** - MIDI-based karaoke files
- **MIDI Support** - Standard MIDI karaoke files
- **Import/Conversion Tools** - Convert existing formats to KAI

## Professional Karaoke Features

### Singer Rotation Management
- **Multi-singer Queue** - Automated rotation system for multiple performers
- **Singer History** - Track what each singer has performed
- **Wait Time Display** - Show estimated wait times for queued singers
- **Singer Profiles** - Store preferred keys/settings per performer

### Song Database Integration
- **Online Karaoke Databases** - Connect to commercial karaoke catalogs
- **Cloud Library Sync** - Sync song collections across devices
- **Metadata Enhancement** - Auto-fetch song info, lyrics, artwork
- **Community Sharing** - Share custom KAI files with other users

### Mobile App Integration
- **Remote Song Requests** - Allow audience to request songs via phone
- **Mobile Queue Management** - Singers can add themselves to rotation
- **Song Preview** - Let users preview available songs on mobile
- **QR Code Integration** - Easy mobile access to song catalog

## Additional Professional Features
- **Backup/Restore** - Complete system state backup
- **Multi-screen Support** - Separate performer and audience displays
- **Show Templates** - Pre-configured setups for different venues
- **Analytics** - Track popular songs, singer preferences, venue stats
